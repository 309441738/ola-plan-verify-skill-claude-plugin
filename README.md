# Plan Verify Skill · Claude Code 插件

Plan Verify Skill —— 由 CruiseSkillBridge 一键发布的Skill。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-plan-verify-skill-claude-plugin
/plugin install ola-plan-verify-skill@ola-plan-verify-skill-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Plan Verify Skill**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
